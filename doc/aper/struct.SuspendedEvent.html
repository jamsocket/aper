pub struct SuspendedEvent<Transition: PartialEq + Clone> {
    pub time: DateTime<Utc>,
    pub transition: Transition,
}

Represents a transition that a crate::StateMachine would like to receive in the future from the server, as well as the time that it would like to receive it. Each state machine can have either zero or one transition suspended at any point in time, and may change or remove that suspended event every time its state is updated (and only then).

Storing and maintaining the suspended events is the responsibility of the code that owns the crate::StateMachine object.

Fields

time: DateTime<Utc>

When the event should be triggered. Note that this is not necessarily equal to the timestamp field on the crate::TransitionEvent that is created when this event is triggered, since there may be a small delay between the time an event is requested at and the time it is actually fired. The latter is used as the timestamp when creating the crate::TransitionEvent.

transition: Transition

The transition that should be triggered. This is turned into a crate::TransitionEvent instance using None as the player, since these events are not considered to be initiated by any player.
